GitHub

ユーザーが自転車で走行する上で危険な箇所をGoogle Map上に登録し、登録内容をシェアするサービスです。

工夫した点

  • Google Map APIを用いるだけではなく、ユーザーがマップ上でクリックした箇所にマーカーを置き、保存時にはそのままマーカーの緯度・経度が保存されるようにしている点。
  • データベース設計時に正規化し、細かくテーブルを分けた上で、データを登録する際には同時に親子関係にあるモデルを同時保存できるようした点。
  • ユーザーが使う状況(電波が弱い可能性等)を想定し、過度な装飾を避け、シンプルなデザインにした点。

これからの課題

  • テストコードが不十分な点。
  • 投稿を削除するとき、確認ダイアログが実装されていない点。

開発期間:約3ヶ月(現在も開発継続中)

開発環境及び使用技術

開発環境: Mac OS Catalina, Editor: Vim

使用技術: HTML/CSS/Bootstrap/JavaScript/jQuery/Ruby/Ruby on Rails/MySQL/AWS(EC2にデプロイ)
